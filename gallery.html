SUNSITE : GALLERY

SOFTWARE RASTERIZER

A polygon-rasterization renderer written in standard C. I have written small libraries for reading/writing BMP images, loading Wavefront OBJ meshes, and performing the linear algebra needed for many basic rendering techniques. This renderer makes great use of these libraries of mine, and was written partly to test them out. The project is still in-progress, and I am actively working on it. Many of the basic concepts and much of the math applied in this project are based on those found in textbooks such as Eric Lengyel's "Foundations of Game Engine Development" series.


ECS ENGINE WITH OPENGL

An old attempt at an ECS-architecture engine. Mesh and texture importing, OpenGL/GLSL rendering. Interactibility and basic kinematic physics via entities, components and systems. I have dropped this project, but I plan on starting up something similar soon.


BMP TOOLS AND TOYS

To test my BMP library, I wrote a small series of image generation and processing programs. Shown here are images from my Kuwahara filtering, Floyd-Steinberg dithering, and Mandelbrot set rendering programs. I referenced Wikipedia articles to write these programs. As I hear of more interesting image processing techniques, I will likely continue to write programs like these.


DEVIL QUELLER: DELONE TRIANGULATION AND R-TREES

My largest current hobby project is a 3D FPS called Devil Queller. I am building it in Unity, but I don't let that stop me from doing lots of experimentation and wheel-reinvention. Shortly after getting some basic FPS gameplay going, I began reading about navmeshing to see if I could generate navmeshes for my levels. I played around with spatial indexing via R-trees and mesh generation via Delone triangulation. In the end, I was pulled away from Devil Queller by schoolwork. While I do not have a schedule for completing this game like I usually do, it remains in-progress and I chip away at it when I have the time.


CATALOGUE

Catalogue is a 2D game that I developed solo for my school's game development club. In addition to doing all of the design and programming, I made all of the art with ink and paper. It's a very imperfect game, but I hope it is an interesting and sometimes delightful one.
