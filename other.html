Some of my projects which have not been given a dedicated page on my website:

  • BMP image reading/writing library written in standard C, with support for memory-mapped I/O
  • Image processing demos in employing my BMP image library, including implementations of Kuwahara filtering and Floyd-Steinberg dithering
  • Image generation demos employing either my BMP image library or the language Processing, including Mandelbrot set rendering and chaos game fractal generation
  • Linear algebra library written in standard C, with no dynamic allocation, supporting common matrix and vector operations on matrices of arbitrary dimension up to a fixed total size
  • Regex parsing library written in standard C, which can determine if strings match regex given in infix notation and can create graphviz graphs of the NFA for a given regex
  • Digital and traditional work in life drawing, packaging design, illustration, and general graphic design
